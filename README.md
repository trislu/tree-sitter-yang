## Introduction

A [YANG](https://www.rfc-editor.org/rfc/rfc6020) parser library based on [tree-sitter](https://tree-sitter.github.io/tree-sitter/)

## The `YANG` RFCs
+ https://www.rfc-editor.org/rfc/rfc6020
+ https://www.rfc-editor.org/rfc/rfc7950 (version 1.1)
+ https://www.rfc-editor.org/rfc/rfc9890
